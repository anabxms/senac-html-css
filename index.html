Inovação No Ensino

HTML5

HTML5 é a mais recente evolução do padrão que define o HTML. Esta nova versão traz consigo importantes mudanças quanto ao papel do HTML no mundo da Web, através de novas funcionalidades como semântica e acessibilidade. Possibilita o uso de novos recursos antes possíveis apenas com a aplicação de outras tecnologias.

CSS3

CSS3 é a terceira e mais nova versão do CSS, onde se definem estilos para seu projeto web. Podemos citar entre suas novas capacidades de formatação visual, as queridas bordas redondas, o uso de sombras, múltiplos backgrounds, layouts multi-colunas, novos seletores, uso de fontes externas, transformações 2D e 3D e animações.

JavaScript

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível e multi-paradigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.

Quem Somos

Nosso propósito é ensinar, criar novas metodologias e ajudar outras pessoas a se tornar novos profissionais de tecnologia e desenvolvimento. Fazemos isso porque acreditamos que o ensino é uma arte, que deve ser feita com paixão, dedicação e organização. Queremos levar a nossa metodologia e os nossos conceitos ao maior número possível de pessoas, alunos, clientes, projetos e desenvolvedores. Contribuindo com a comunidade tecnológica disseminando conhecimento. Fazemos isso através de palestras, consultorias, treinamentos, vídeo aulas e conteúdo em redes sociais e livros.
